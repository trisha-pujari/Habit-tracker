# HabitAI  AI-Powered Habit Tracker

Developed a full-stack AI Habit Tracker using MERN, JWT authentication,
Tailwind CSS, and Google Gemini AI featuring habit tracking, streak
analytics, 90-day heatmap, AI insights, personalized recommendations,
and responsive UI.
---

## Features

- **Habit Tracking**  Create, manage, and log daily/weekly habits with streak tracking
- **Weekly Report**  AI-generated personalised summary of your habit performance over the past 7 days
- **Habit Suggestions**  AI recommends 3 habits tailored to your goals, schedule, and struggles
- **Recovery Plan**  Broke a streak? Get a warm, actionable 3-day plan to get back on track
- **Chat Analysis**  Ask natural language questions about your habits and get data-driven answers
- **Morning Motivation**  Start every day with a personalised motivational message based on your current streaks

---

## Tech Stack

| Layer | Technology |
|---|---|
| Runtime | Node.js |
| Framework | Express.js |
| Database | MongoDB + Mongoose |
| AI | Google Gemini API |
| Auth | JWT |
